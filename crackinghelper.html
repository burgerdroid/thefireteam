Cracking Helper

Instructions

This is a crude tool that will assist in getting more correct guesses when doing the Cracking crime (there is a merit for getting 250 correct guesses). Sometimes passwords will contain numbers, or not be a dictionary word. In those cases this tool will not be able to assist.

  1. Set the password length to the size of the password you are trying to guess.
  2. Make a crack attempt and put the new letter into the corresponding character box.
  3. Click Get Candidate Words to reduce the search space and show the top 10 highest frequency letters for each character.
  4. When making a guess in the Cracking crime:
    • If it is correct, add the new correct letter to the corresponding character box.
    • If it is incorrect, delete the letter from the A-Z text field.
    • After either of these 2 scenarios, click again on Get Candidate Words
